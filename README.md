# AI Powered Recruitment Tool

## Features
- Recruiter authentication
- Job posting
- Resume parsing
- AI-based candidate matching

## Tech Stack
Frontend: React, Tailwind  
Backend: Node.js, Express  
AI: LLM / NLP model  
DB: MongoDB  
Deployment: Vercel + Render

## Setup
1. Clone repo
2. Add env variables
3. npm install
4. npm start

## Deployment
Frontend: Vercel  
Backend: Render

## AI Tools Used
- LLM for resume parsing
- Embeddings for similarity matching
